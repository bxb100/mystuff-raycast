This is a simple worker to handle the upload and download of image files with R2

It is used to replace the https://google-lens.yencheng.dev functionality

Remember to change CORS and Delete Rules in the R2 setting

```json
[
  {
    "AllowedOrigins": [
      "https://google.com"
    ],
    "AllowedMethods": [
      "GET"
    ]
  }
]
```
